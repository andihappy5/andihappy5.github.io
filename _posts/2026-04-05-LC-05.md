## 题目描述
最长的回文字符串   

Given a string s, return the longest palindromic substring in s.

 

Example 1:

Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.
Example 2:

Input: s = "cbbd"
Output: "bb"
 

Constraints:

1 <= s.length <= 1000
s consist of only digits and English letters.

## two pointers key points
- 维护两个指针，分别指向开始和结束位置
- 通过移动指针来发现具体的答案

~~~java  
class Solution {
        public String longestPalindrome(String s) {
            // deal special case
            if (s == null || s.length() < 2) {
                return s;
            }
            int len = 1;
            int index = 0;
            // beacse the longest palindrome substring must be less than or equal to the
            // length of s, the loop can be from 0 to s.length()
            for (int i = 0; i < s.length(); i++) {
                int length = isPalindrome(s, i);
                if (length > len) {
                    len = length;
                    index = i;
                }
            }
            if (len % 2 == 0) {
                return s.substring(index - (len / 2 - 1), index + len / 2+1);
            } else {
                return s.substring(index - len / 2, index + len / 2 + 1);
            }
        }

        private int isPalindrome(String s, int i) {
            // there are two cases, one is the length of the palindrome substring is odd,
            // the other is the length of the palindrome substring is even
            int len1 = 1;
            int len2 = 0;
            // case 1: the length of the palindrome substring is odd
            for (int j = 1; i - j >= 0 && i + j < s.length(); j++) {
                if (s.charAt(i - j) == s.charAt(i + j)) {
                    len1 += 2;
                } else {
                    break;
                }
            }
            // case 2: the length of the palindrome substring is even
            for (int j = 0; i - j >= 0 && i + j + 1 < s.length(); j++) {
                if (s.charAt(i - j) == s.charAt(i + j + 1)) {
                    len2 += 2;
                } else {
                    break;
                }
            }
            return Math.max(len1, len2);
        }
    }
~~~   
## Dynamic Programming
其实还有一种动态规划的场景，这个场景算是一种复习吧，作为动态规划的入门题目，还是很不错的。动态规划的核心思想是通过维护一个二维数组来记录子问题的解，从而避免重复计算。对于这个问题，我们可以定义一个二维布尔数组 `dp[i][j]`，表示字符串从索引 `i` 到索引 `j` 是否是回文字符串。

// 那么我们先初始化这个数组
i=j 的时候，dp[i][j] = true，因为单个字符一定是回文字符串
dp[i][j] = true 当 s.charAt(i) == s.charAt(j) && dp[i+1][j-1] == true
我们需要从长度为1的子字符串开始，逐渐增加子字符串的长度，直到达到字符串的长度。每次更新 dp 数组时，我们还需要记录最长的回文字符串的起始位置和长度，以便最后返回结果。   

~~~java
class Solution {    
    public String longestPalindrome(String s) {
         if (s == null || s.length() < 2)
            return s;
        int len = s.length();
        int max = 0, from = 0, to = 0;
        boolean[][] dp = new boolean[len][len];
        for (int j = dp.length - 1; j >= 0; j--) {
            for (int i = j; i < dp.length; i++) {
                dp[j][i] = (i - j <= 1 && s.charAt(j) == s.charAt(i))
                        || (dp[j + 1][i - 1] && s.charAt(j) == s.charAt(i));
                if (dp[j][i] && (i - j + 1 >= max)) {
                    max = i - j + 1;
                    from = j;
                    to = j2;
                }
            }
        }
        return s.substring(from, to + 1);
    }
} 
~~~
