7. Reverse Integer

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0.

Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

 

Example 1:

Input: x = 123
Output: 321
Example 2:

Input: x = -123
Output: -321
Example 3:

Input: x = 120
Output: 21
 

Constraints:

-231 <= x <= 231 - 1

## 解题思路

~~~java
    public int reverse(int x) {
        // deal corner case
        boolean flag = x > 0 ? true : false;
        x = Math.abs(x);
        int res = 0;
        while (x > 0) {
            int tmp = x % 10;
            if ((Integer.MAX_VALUE - tmp) / 10 < res) {
                return 0;
            }
            res = 10 * res + tmp;
            x = x / 10;
        }
        return flag ? res : -res;
    }
~~~

## English explain words
Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range, then return 0.

Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

The key points to consider are:
1. The input integer can be positive or negative.
2. If the reversed integer exceeds the 32-bit signed integer range, we should return 0.
3. We need to handle the reversal without using 64-bit integers.

My approach is to first determine the sign of the input integer and work with its absolute value. Then, I will reverse the digits by repeatedly taking the last digit and building the reversed number. During this process, I will check for potential overflow before updating the reversed number. Finally, I will return the reversed number with the correct sign.

Time complexity is O(log10(n)) where n is the input integer, because we are processing each digit of the number. Space complexity is O(1) since we are using a constant amount of space for the variables.

8. String to Integer (atoi)

Implement the myAtoi(string s) function, which converts a string to a 32-bit signed integer.

The algorithm for myAtoi(string s) is as follows:

Whitespace: Ignore any leading whitespace (" ").
Signedness: Determine the sign by checking if the next character is '-' or '+', assuming positivity if neither present.
Conversion: Read the integer by skipping leading zeros until a non-digit character is encountered or the end of the string is reached. If no digits were read, then the result is 0.
Rounding: If the integer is out of the 32-bit signed integer range [-231, 231 - 1], then round the integer to remain in the range. Specifically, integers less than -231 should be rounded to -231, and integers greater than 231 - 1 should be rounded to 231 - 1.
Return the integer as the final result.

Input: s = "42"

Output: 42

         ^
Step 3: "42" ("42" is read in)
           ^

Explanation:

Step 1: "   -042" (leading whitespace is read and ignored)

             ^
